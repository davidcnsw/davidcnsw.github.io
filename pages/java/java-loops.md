Eclipse example: loops

Java has three kinds of loops:

**while loop**

```java
while (condition) {
// run
}
```

**do...while loop**

```java
do {
// run
}
while(condition);
```

**for Loop**

```java
for(counter initial value, condition, post loop action)
```
