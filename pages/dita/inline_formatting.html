For a cascade of actions, use menucascade in combination with uicontrol: {% highlight xml %}To open the document, go to FileOpen.{% endhighlight %}

By using menucascade and uicontrol, you can implement different formatting than plain bold (although by default, it will look the same in Oxygen's webhelp output).

{{tip}} It's sometimes easier to add inline formatting via the Author view in Oxygen. Just highlight a word and press Enter. Then type the element you want to apply. The next time you highlight a word and press Enter, Oxygen prompts you with the last element you applied. Essentially you can highlight a word and press Enter twice to quickly apply formatting to groups of elements. This is especially helpful when you're applying codeph tags.{{end}}

Note that you cannot use strong instead of b.

You can also use basic HTML list tags, such as ol and ul. You can use these list tags in a concept.
